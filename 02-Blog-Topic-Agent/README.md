# 🤖 Local AI Blog Generator Agent

## Overview

This project is a Local AI Agent built using Ollama, Gemma 3, and Streamlit. It generates trending blog topics for an AI services company and automatically creates professional blog content using a locally running Large Language Model (LLM).

---

## Features

- Local AI Agent using Ollama
- Gemma 3 LLM integration
- Company-based blog topic generation
- Automatic blog writing
- Streamlit user interface
- Modular project structure

---

## Technologies Used

- Python
- Streamlit
- Ollama
- Gemma 3 (Local LLM)
- Requests
