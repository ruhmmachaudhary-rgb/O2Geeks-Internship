#  Local AI Agent using Ollama & Streamlit

##  Project Overview

This project is a simple Local AI Agent built using Streamlit and Ollama. The AI model runs completely on the local machine without using any cloud API or paid services.

---

##  Features

- Local AI Agent
- Ollama Integration
- Streamlit User Interface
- Real-time AI Responses
- Offline Execution
- Lightweight and Beginner Friendly

---

##  Technologies Used

- Python
- Streamlit
- Ollama
- Requests Library
- Gemma 3 (1B) Model

##  Installation

### Clone Repository

```bash
git clone <repository-url>
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Environment

Windows

```bash
venv\Scripts\activate
```

### Install Dependencies

```bash
pip install streamlit requests ollama
```

### Run Ollama

```bash
ollama run gemma3:1b
```

### Run Streamlit

```bash
streamlit run app.py
```

##  Output

The application provides a simple interface where users can enter a question and receive AI-generated responses locally using the Gemma model.
